ToTheMoon is a Bitcoin web faucet video game designed by the MoonBase Games team:

##Jordan Frankfurt: Co-founder & CEO
Jordan's programming focus:
  *IO.JS/Node.js
  *Express
  *MongoDB

##Ryan Skinner: Co-founder & CTO

##Phillip Chaffee: Web Developer
